# weather-command-line-rs

A weather command line tool build by Rust.
一个使用 Rust 编写的获取天气命令行工具。

## 使用

cargo run <城市名拼音>

```shell
cargo run hangzhou
```

控制台将会输出

```md
-----正在获取天气信息, 请稍后...-----
当前温度：25.95℃
今日最低温：25.95℃
今日最高温：25.95℃
体感温度：25.95℃
湿度：84%
日出时间：05:03:06
日落时间：18:48:51
所在经度：120.1614
所在纬度：30.2937
```
